# Wati Sandbox Personas

The AI MUST use one of these six fictional Wati customers when 
generating mock data for previews. Pick the persona that best matches 
the prompt context. If no clear match, default to Priya's Saree Studio.

DO NOT invent new customers. DO NOT use generic names like "John Doe", 
"Acme Corp", or "user_001".

## Usage rules

When generating mock data for previews:
- Pick the persona that fits the prompt context
- Use their actual numbers (contacts, open rates, etc.)
- Use their template names and broadcast titles
- Use realistic language fragments from their primary language 
  (a few words for flavor, not full localization)
- If the prompt mentions a specific use case, pick the persona whose 
  use cases match
- If unclear, default to Priya's Saree Studio
